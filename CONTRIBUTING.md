# Contributing to CholesTrack

Thank you for your interest in improving CholesTrack!

## 🛣 Roadmap
- [ ] Real Firebase Auth implementation.
- [ ] Firestore persistence for multi-device sync.
- [ ] Export data to CSV/JSON.
- [ ] Advanced "What-if" simulations for fiber/exercise impact.

## 🤝 How to Contribute
1. **Fork the Repository**: Create your own branch from `main`.
2. **Local Development**:
   - Use a simple HTTP server to run the app.
   - Modify `app.js` or `index.html`.
3. **Coding Standards**:
   - Use semantic HTML5.
   - Follow the Tailwind class order (layout -> spacing -> typography -> colors).
   - Comment heavily on any complex data transformation logic.
4. **Submit a PR**: Provide screenshots/videos of your changes.

## 🎨 Design System
- **Background**: `#0a0a0c`
- **Primary Accent**: Emerald-500 (`#10b981`)
- **Secondary Accent**: Blue-500 (`#3b82f6`)
- **Font**: Inter (UI) and JetBrains Mono (Data/Code).
